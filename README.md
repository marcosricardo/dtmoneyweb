## Starting

```bash
# Clone this project
$ git clone https://github.com/marcosricardo/dtmoneyweb

# Access
$ cd dtmoneyweb

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
